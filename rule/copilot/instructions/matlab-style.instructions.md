# MATLAB 代码风格规范

## 文件头模板

每个 `.m` 文件顶部须包含：

```matlab
% =========================================================================
% 文件    : xxx.m
% 描述    : 模块简要说明
% 版本    : v1.0
% 日期    : YYYY/MM/DD
%
% 修改记录（最新版本在最前）:
%  ver     date        modification
% ------  ----------  -----------------------------------------------------
%  v1.0   YYYY/MM/DD  创建文件
% =========================================================================
```

- 修改记录仅保留最近 **3** 个版本，最新在最前

## 函数注释

函数文件紧接文件头，用 `%` 注释说明用途、输入、输出：

```matlab
function result = calc_fft(signal, fs)
% calc_fft  计算信号的 FFT 频谱
%
% 输入：
%   signal  - 时域信号向量
%   fs      - 采样率（Hz）
%
% 输出：
%   result  - 频谱结构体，含 freq（频率轴）和 amp（幅值）
```

## 注释规范

- 注释使用**中文**，行内注释用 `%` 跟随代码
- 区块标题用 `%% 内容`（产生代码折叠节）
- 新增代码标注版本与日期：`% vX.Y YYYY/MM/DD 新增：说明`

## 其他约定

- 避免使用 `clear all` / `close all`（视需要在脚本顶部显式清理）
- 矩阵运算优先于循环；必须用循环时加注释说明原因
- 图形输出须设置标题、坐标轴标签和单位
