## 5. 代码通用规范

- 修改函数/接口时须说明对其他模块的**影响范围**
- 废弃代码添加注释说明原因，保留 **3个版本** 后清理
- 新增代码在行尾或上方注释标注日期：`// YYYY-MM-DD 新增：说明`

---

## 6. C/C++ 代码规范

- **4个空格**缩进，禁止 Tab；大括号 K&R 风格（左括号不换行）
- 函数调用**必须写在同一行**，禁止拆分参数到多行
- 行尾注释用 `//`，禁止用 `/* */`（含多行注释，文件头除外）；区块标题用 `// ---- 内容 ----//`
- 同一连续代码块（结构体字段、成组宏、初始化表项）的行尾注释**纵向对齐**；对齐影响可读性时，仅在本次触及的局部块内对齐
- 函数头使用 Doxygen 风格（中文）
- 文件头模板（`.c` / `.h`）：
  ```c
  /*
   * 文件 : xxx.c
   * 描述 : 模块简要说明
   * 版本 : v1.0
   * 日期 : YYYY-MM-DD
   *
   * 修改记录（最新版本在最前）:
   *  ver  who      date       modification
   * ----- -------  ---------- ---------------------------------
   * 1.0   xxx      YY/MM/DD   创建文件
   */
  ```
- 修改记录仅保留最近 **3** 个版本

---

## 7. Qt C++ 补充规范

- `QWidget` / `QMainWindow` 只做 UI 更新，不阻塞
- 耗时操作放入 `QThread` 子类；跨线程**只用信号槽**（`Qt::QueuedConnection`）
- 继承 `QObject` 的类第一行包含 `Q_OBJECT`，构造函数接受 `QObject *parent = nullptr`
- 命名：信号用动词过去式（`dataReceived`），槽用 `on` + 信号名（`onDataReceived`），UI 成员用 `m_` 前缀

---

## 8. 命名规范

**C/C++**：变量/函数用小写下划线，宏用全大写下划线，全局变量加 `g_` 前缀，类名大驼峰

**Python**：变量/函数小写下划线，常量全大写下划线，类名大驼峰，私有成员单下划线前缀

**Verilog**：模块名/信号名小写下划线，参数/宏全大写下划线，寄存器加 `_r` 后缀，复位低有效用 `rst_n`

通用：布尔变量用 `is_`、`has_`、`can_` 前缀；避免单字母命名（循环变量除外）

---

## 9. Verilog / SystemVerilog 代码规范

- **2个空格**缩进，禁止 Tab
- `begin` 必须**另起一行**，不得与 `always`/`if`/`else`/`for` 写在同一行
- 单bit信号必须显式写位宽：`1'b0` / `1'b1`，禁止直接用 `0` / `1`
- 时序逻辑用 `<=`，组合逻辑用 `=`；端口声明每行一个信号，对齐排列
- 文件头模板（`.v`）：
  ```verilog
  /**************************************************************************/
  // Function   : 模块功能简述
  // Version    : v1.0
  // Date       : YYYY/MM/DD
  // Description: 详细说明
  //
  // Modify:
  // version       date       modify
  // --------    -----------  ------------------------------------------------
  //  v1.0        YYYY/MM/DD  创建文件
  /**************************************************************************/
  ```

## 10. Python 规范

- 不使用虚拟环境，直接用系统全局 Python；安装依赖用 `pip install`
- 文件头模板：
  ```python
  """
  文件名.py  --  模块简要说明
  版本    : vX.Y
  日期    : YYYY-MM-DD

  修改记录:
      vX.Y  新增/修复/调整 说明
  """
  ```
- 注释使用中文；新增代码标注日期 `# YYYY-MM-DD 新增：说明`
- PyQt5：GUI 线程只做 UI 更新，工作线程独立维护资源，跨线程只用信号槽

## 11. 架构要点

**PS 端（Vitis）**：init + poll 模式；Log 四级分层（`prinfo`/`prwarn`/`prerror` 常开，`prbrief` 默认开，`prdebug`/`prlist`/`prhigh` 默认关）

**协议命令**：`CmdType = 0xAABBCCDD`（AA=执行类型，BB=大类，CC=子命令，DD=应答类型）

**DDR 布局**：代码区 → PROG(32MB) → CORR(8MB) → JUMP(4KB) → DMA_X/Y/Z(各10MB) → LIST(16MB)

**PL 端（Verilog）**：时序用 `<=`，组合用 `=`；复位统一命名（低有效 `rstn`，高有效 `rst`）
