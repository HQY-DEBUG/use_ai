## 概述

本文档列出 `copilot` 命令在**启动阶段**（命令行中直接传入）可用的子命令与选项，不含会话内斜杠命令（`/command`）。

---

## 子命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `copilot` | 启动交互式会话 | `copilot` |
| `copilot help [topic]` | 显示帮助，可指定主题：`config`/`commands`/`environment`/`logging`/`permissions` | `copilot help permissions` |
| `copilot init` | 为当前仓库初始化自定义指令 | `copilot init` |
| `copilot update` | 下载并安装最新版本 | `copilot update` |
| `copilot version` | 显示版本号并检查更新 | `copilot version` |
| `copilot login` | 通过 OAuth 设备流进行身份验证 | `copilot login` |
| `copilot logout` | 登出并移除凭据 | `copilot logout` |
| `copilot plugin` | 管理插件和市场 | `copilot plugin list` |

## 启动选项

### 执行模式

| 选项 | 说明 | 示例 |
|------|------|------|
| `-p PROMPT`, `--prompt=PROMPT` | 以编程方式执行提示词后退出（非交互） | `copilot -p "重构这个函数"` |
| `-i PROMPT`, `--interactive=PROMPT` | 携带初始提示词启动交互式会话并自动执行 | `copilot -i "解释项目结构"` |
| `--continue` | 恢复最近一次会话 | `copilot --continue` |
| `--resume=SESSION-ID` | 恢复指定 ID 的会话 | `copilot --resume=abc123` |
| `--autopilot` | 启用自主继续模式（自动执行多步任务） | `copilot --autopilot` |
| `--acp` | 启动 Agent Client Protocol 服务器 | `copilot --acp` |

### 权限与访问控制

| 选项 | 说明 | 示例 |
|------|------|------|
| `--allow-all` | 启用所有权限（等同于 `--yolo`） | `copilot --allow-all` |
| `--yolo` | 启用所有权限（`--allow-all` 的别名） | `copilot --yolo` |
| `--allow-all-paths` | 禁用文件路径验证，允许访问所有路径 | `copilot --allow-all-paths` |
| `--allow-all-tools` | 允许所有工具无需确认 | `copilot --allow-all-tools` |
| `--allow-all-urls` | 允许访问所有 URL | `copilot --allow-all-urls` |
| `--allow-tool=TOOL` | 允许特定工具（可多次指定） | `copilot --allow-tool=shell --allow-tool=write` |
| `--allow-url=URL` | 允许特定 URL 或域名（可多次指定） | `copilot --allow-url=github.com` |
| `--deny-tool=TOOL` | 阻止特定工具（可多次指定） | `copilot --deny-tool=shell` |
| `--deny-url=URL` | 阻止特定 URL（可多次指定） | `copilot --deny-url=example.com` |
| `--add-dir=PATH` | 将目录添加到允许的文件访问列表 | `copilot --add-dir=./src` |

> 工具权限使用 `Kind(argument)` 格式，例如 `shell(git push)`、`write(src/*.ts)`、`url(github.com)`。

### 模型与 AI 配置

| 选项 | 说明 | 示例 |
|------|------|------|
| `--model=MODEL` | 设置本次会话使用的 AI 模型 | `copilot --model=gpt-4o` |
| `--agent=AGENT` | 指定自定义 Agent | `copilot --agent=my-reviewer` |

### 工具配置

| 选项 | 说明 | 示例 |
|------|------|------|
| `--available-tools=TOOL` | 限制本次会话可用的工具列表（可多次指定） | `copilot --available-tools=shell --available-tools=read` |
| `--excluded-tools=TOOL` | 排除特定工具（可多次指定） | `copilot --excluded-tools=write` |
| `--disable-parallel-tools-execution` | 禁用工具并行执行 | `copilot --disable-parallel-tools-execution` |

### MCP 配置

| 选项 | 说明 | 示例 |
|------|------|------|
| `--disable-builtin-mcps` | 禁用所有内置 MCP 服务器 | `copilot --disable-builtin-mcps` |
| `--disable-mcp-server=SERVER-NAME` | 禁用特定 MCP 服务器 | `copilot --disable-mcp-server=filesystem` |
| `--additional-mcp-config=JSON` | 为本次会话添加 MCP 服务器（JSON 格式） | `copilot --additional-mcp-config='{"name":"my","command":"..."}' ` |
| `--add-github-mcp-tool=TOOL` | 启用指定的 GitHub MCP 工具 | `copilot --add-github-mcp-tool=create_issue` |
| `--add-github-mcp-toolset=TOOLSET` | 启用指定的 GitHub MCP 工具集 | `copilot --add-github-mcp-toolset=issues` |
| `--enable-all-github-mcp-tools` | 启用所有 GitHub MCP 工具 | `copilot --enable-all-github-mcp-tools` |

### 输出与显示

| 选项 | 说明 | 示例 |
|------|------|------|
| `-s`, `--silent` | 仅输出 Agent 响应，不显示其他信息 | `copilot -p "query" --silent` |
| `--output-format=FORMAT` | 输出格式：`text`（默认）或 `json`（JSONL） | `copilot -p "query" --output-format=json` |
| `--no-color` | 禁用彩色输出 | `copilot --no-color` |
| `--plain-diff` | 禁用富文本差异渲染 | `copilot --plain-diff` |
| `--banner` | 显示启动横幅 | `copilot --banner` |
| `--screen-reader` | 启用屏幕阅读器优化 | `copilot --screen-reader` |
| `--stream=MODE` | 启用或禁用流式传输 | `copilot --stream=false` |
| `--alt-screen=VALUE` | 使用终端备用屏幕 | `copilot --alt-screen=true` |

### 配置文件与日志

| 选项 | 说明 | 示例 |
|------|------|------|
| `--config-dir=PATH` | 指定配置目录（覆盖默认路径） | `copilot --config-dir=~/.my-copilot` |
| `--log-dir=DIRECTORY` | 指定日志文件目录 | `copilot --log-dir=./logs` |
| `--log-level=LEVEL` | 设置日志详细程度 | `copilot --log-level=debug` |
| `--no-custom-instructions` | 禁用自定义指令文件加载 | `copilot --no-custom-instructions` |
| `--experimental` | 启用实验性功能 | `copilot --experimental` |
| `--no-experimental` | 禁用实验性功能 | `copilot --no-experimental` |

### 自动驾驶与行为控制

| 选项 | 说明 | 示例 |
|------|------|------|
| `--max-autopilot-continues=COUNT` | 限制自动驾驶模式最大消息轮次 | `copilot --autopilot --max-autopilot-continues=10` |
| `--no-ask-user` | 禁用所有用户提示（完全自动化场景） | `copilot -p "query" --no-ask-user` |
| `--no-auto-update` | 禁用自动更新检查 | `copilot --no-auto-update` |

### 会话分享

| 选项 | 说明 | 示例 |
|------|------|------|
| `--share=PATH` | 会话结束后将结果共享到指定 Markdown 文件 | `copilot -p "query" --share=./result.md` |
| `--share-gist` | 会话结束后共享到私密 GitHub Gist | `copilot -p "query" --share-gist` |

### 安全与隐私

| 选项 | 说明 | 示例 |
|------|------|------|
| `--secret-env-vars=VAR` | 在日志和输出中脱敏指定环境变量（可多次指定） | `copilot --secret-env-vars=API_KEY` |

### 其他

| 选项 | 说明 | 示例 |
|------|------|------|
| `-h`, `--help` | 显示帮助信息 | `copilot --help` |
| `-v`, `--version` | 显示版本信息 | `copilot -v` |

## 环境变量（影响启动行为）

| 变量 | 说明 |
|------|------|
| `COPILOT_GITHUB_TOKEN` | 认证令牌（最高优先级） |
| `GH_TOKEN` | GitHub 令牌（第二优先级） |
| `GITHUB_TOKEN` | GitHub 令牌（最低优先级） |
| `COPILOT_MODEL` | 设置默认 AI 模型 |
| `COPILOT_ALLOW_ALL` | 启用所有权限（设为 `true`） |
| `COPILOT_AUTO_UPDATE` | 禁用自动更新（设为 `false`） |
| `COPILOT_HOME` | 覆盖配置和状态目录 |
| `COPILOT_CUSTOM_INSTRUCTIONS_DIRS` | 追加自定义指令目录 |
| `COPILOT_SKILLS_DIRS` | 追加技能目录 |
| `COPILOT_EDITOR` | 外部编辑器命令（`Ctrl+G` 调用） |
